如何运行 Lumina 博客代码

这段代码是一个 React 单页应用（SPA），依赖于以下技术栈：

React: UI 框架

Tailwind CSS: 样式引擎（那堆 bg-slate-950 类的来源）

Framer Motion: 动画库（负责丝滑的转场和悬停效果）

Lucide React: 图标库

你可以通过以下方式运行：


如果你想把它作为一个真正的个人网站项目保存下来，请按照以下步骤操作：

1. 准备工作

确保你的电脑安装了 Node.js (推荐 v18 或更高版本)。

2. 创建项目

打开终端（Terminal 或 CMD），运行以下命令创建一个基于 Vite 的 React 项目：

npm create vite@latest my-blog -- --template react
cd my-blog


3. 安装 Tailwind CSS

这是最关键的一步，没有它页面会是一团乱。运行：

npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p


然后打开项目中的 tailwind.config.js，修改 content 部分：

/** @type {import('tailwindcss').Config} */
export default {
  content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [
    require('@tailwindcss/typography'), // 可选：如果你想让文章排版更好看，建议安装这个插件
  ],
}


注意：如果上面的配置报错，你可能需要运行 npm install -D @tailwindcss/typography 来安装排版插件，或者先在配置里删掉 plugins 那一行。

打开 src/index.css，把里面的内容全部删掉，换成：

@tailwind base;
@tailwind components;
@tailwind utilities;

body {
  background: #020617; /* bg-slate-950 */
}


4. 安装依赖库

安装我代码中用到的图标和动画库：

npm install framer-motion lucide-react clsx tailwind-merge


5. 植入代码

打开项目文件夹中的 src/App.jsx。

清空里面的所有代码。

将我发给你的 LuminaBlog.jsx 中的所有代码完整复制并粘贴进去。

6. 启动！

回到终端，运行：

npm run dev


按住 Ctrl 点击终端里显示的链接（通常是 http://localhost:5173），你就能在浏览器看到你的极高颜值博客了！

常见问题

Q: 为什么页面看起来没有样式，乱糟糟的？
A: 99% 是因为 Tailwind CSS 没有配置好。请仔细检查第 3 步，确保 index.css 里引入了 @tailwind 指令，且 tailwind.config.js 文件存在。

Q: 图片加载不出来？
A: 我使用的是 Unsplash 的在线图片链接。如果你的网络无法访问 Unsplash，图片可能会挂掉。你可以替换代码中 IMAGES 对象里的链接为你自己的图片链接。
